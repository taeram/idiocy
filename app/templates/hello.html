Idiocy

Idiocy is a Python powered URL shortening service.

To shorten a URL, just POST it to {{ url_for('shorten', _external=True ) }} using your API key: 

curl -X POST {{ url_for('shorten', _external=True ) }} -H "Authorization: secret_api_key" -F "url=http://example.com/kitty.gif"

The body of the response will contain the shortened URL: 

{{ url_for('bounce', code="oT0Dh", _external=True )|strip_www }}

Some services will generate image previews of an image URL if it has the correct file extension. To enable this, simply add a file extension to the shortened URL: 

{{ url_for('bounce', code="oT0Dh.gif", _external=True )|strip_www }}
